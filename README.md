## 🛠️ Technical Implementation

- **Dynamic Routing:** Implemented `react-router-dom` to handle navigation between the catalog and product detail pages seamlessly.
- **API Integration:** Used the `Fetch API` combined with React hooks (`useState`, `useEffect`) to asynchronously retrieve and display live product data.
- **State Management:** Mastered the flow of data across components to ensure the UI remains synchronized with user selections.

## 🧠 Key Learnings

This project was instrumental in understanding how to handle large datasets and API responses. I focused on optimizing performance and creating a **Mobile-First** architecture using CSS Grid and Flexbox to ensure a professional look on all devices.
